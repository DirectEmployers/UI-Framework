DEF UI Framework

JavaScript Framework

The most commonly used files are available in the javascript core loader file, def.ui.core.js, which you can use easily by including the following in your <head> section:

<script src="http://js.nlx.org/framework/js/def.ui.core.js"></script>

The DEF javascript framework is built on 3 tiers:

jQuery
jQuery is a JavaScript library that simplifies and accelerates common JavaScript tags. The DEF UI Framework makes use of jQuery, jQueryUI, and jQueryMobile Libraries.
More info at jquery.org
Bootstrap
Bootstrap is primarily a CSS library, and is developed by the engineering team at Twitter. The DEF UI framework uses the full Bootstrap CSS library as a starting point for all display components, and then builds from there to customize for brand and custom components. Bootstrap is fully HTML5 and CSS3 compliant.

In addition to the CSS styling, the Bootstrap javascript and interactive components are also available.

More info at twitter.github.com/bootstrap/
DEF custom functons
Every Framework loader loads the foundation core functions in addition to thrid party libraries.

If you need a more detailed approach, or need to make use of any of the bootstrap automated functionality, then you should use the JavaScript Loader Grid determine which files you need to incude.

CSS Framework

The DEF CSS framework is built on top of bootstrap, and is extremely easy to include. Just cut and paste the following into your <head> section:

<link href="http://css.nlx.org/framework/css/def.ui.css" rel="stylesheet" type="text/css">
